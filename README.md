OVERVIEW
This repository contains the complete PCB design for a 4 bit Magnitude Comparator and Display which happens to be part of a larger multi-block digital logic system developed for the EEE 312 (Digital Electronics) Capstone Project.
The module receives two 4-bit binary numbers (A and B) from an external register block, compares them using a 74LS85 magnitude comparator IC, and visually indicates the result via three dedicated LED outputs:
<img width="557" height="182" alt="image" src="https://github.com/user-attachments/assets/6abf688a-70fb-4183-a13b-a6d18426a4d8" />
This comparator is designed in such a way that, at any given time, its always only one LED that will be ON

DESIGN OVERVIEW

i. IC USED: 74LS85 (4 BIT TTL Magnitude Comparator)
ii. Resistors (rated at 330Ω)
iii. Decoupling capacitor (100nF ceramic) across the VCC and GND for noise filtering
iv. Input connector 8 pin 2.54mm pitch header (J1) for A0 - A3 amd B0-B3 signals
v. Power supply (5v TTL Compatible)
